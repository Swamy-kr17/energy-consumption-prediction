# ⚡ Energy Consumption Prediction

This project predicts energy consumption based on temperature using Machine Learning.

---

## Project Overview
- Input: Temperature (°C)
- Output: Energy Consumption (kWh)
- Model: Linear Regression

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Files
- `Day3ML.ipynb` → Machine learning notebook
- `appliance_energy.csv` → Dataset

---

## Results
- R² Score: ~0.61
- Mean Squared Error: ~0.16
